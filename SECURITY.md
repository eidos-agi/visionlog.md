# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in visionlog, please report it responsibly.

**Email:** daniel@eidosagi.com

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will acknowledge your report within 48 hours and aim to provide a fix within 7 days for critical issues.

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |

## Disclosure Policy

We follow coordinated disclosure. Please give us reasonable time to address the issue before public disclosure.
